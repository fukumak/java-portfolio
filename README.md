# Java学習ポートフォリオ 作業フォルダ

Java学習の成果をポートフォリオサイトに掲載するための作業用フォルダです。

## フォルダ構成

- `projects/` — 学習の一環で作成したJavaプログラム・アプリのソースコードを配置
  - 1プロジェクト1フォルダで整理する（例: `projects/todo-app/`）
- `notes/` — 学習内容のメモ・まとめ（Markdown推奨）
- `certifications/` — 資格取得の証明書や合格実績など
- `assets/` — スクリーンショットや図など、掲載用の画像素材
- `portfolio-entry.md` — ポートフォリオサイトに掲載する文章のドラフト

## 進め方（案）

1. `projects/` に既存または今後作成するJavaプログラムを追加
2. `notes/` に学んだ内容（言語仕様、フレームワーク、設計パターンなど）を記録
3. 資格を取得したら `certifications/` に情報を追加
4. 上記を踏まえて `portfolio-entry.md` を執筆
5. 内容が固まったらポートフォリオサイト（HTML/CMS等）に反映
